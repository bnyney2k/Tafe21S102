Welcome To Start Finance website


Start Finance is a repository set up for illustration of TortoiseGit and GitHub functions.

The new development is required to include extra functions to maintain the Personal Information,

Contact Details, Appointments & Shopping List. Each group member is responsible to develop one

of these functional areas （Personal, ContactDetails, Appointment, Shopping List）.

You can fork/clone this repo and add rebuild the application.


Directions

Step 1:

Fork and clone this repository.

Step 2:

Make your new branch(Personal, ContactDetails, Appointment, Shopping List).

Step 3:

Open the StartFinanceMaster with Visual 2019 and add functions to maintain the

Personal Information, Contact Details, Appointments & Shopping List.


Step 4:

Each member update your feature.

Step5:

Merge all features to main branch and push to GitHub.
